Some things I've been working on...

Alpha Kappa Psi Website

As VP-Operations of Alpha Kappa Psi at the University of Michigan, I'm responsible for the chapter website. Written in EJS, HTML and CSS, I make changes & updates to the site, in addition to building out new modules. AKPsi's website is the first thing that prospective members see, and I've been able to understand the importance of having a great UI/UX.

Alpha Kappa Psi Rush Application

A Python Flask/ReactJS application that supports 200-300 rush applicants each semester for Alpha Kappa Psi. My work on this site has been as part of my role on AKPsi's Technology Committee. I'm responsible for much of the front-end, which involves compiling our React components and writing AJAX requests to the API endpoints the back-end team wrote.

A note: due to chapter policy, the source code for this application cannot be made public. It can be provided upon request.
